# CSE447 — Cryptography and Cryptanalysis
> BRAC University | Jupyter Notebook (Python)

Lab assignments covering core cryptographic algorithms and techniques,
from classical ciphers to modern encryption and steganography.

## Labs

| Lab | Topic |
|-----|-------|
| Lab 1 | Caesar Cipher & Substitution Cipher |
| Lab 2 | DES (Data Encryption Standard) |
| Lab 3 | AES (Advanced Encryption Standard) |
| Lab 4 | CBC-MAC & A5/1 Stream Cipher |
| Lab 5 | RSA (Public Key Cryptography) |
| Lab 6 | Hashing (MD5 / SHA) |
| Lab 8 | Steganography |

## Tech Stack
`Python` `Jupyter Notebook` `PyCryptodome`
